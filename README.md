# 一航个人网站 v0.2

React + Vite 版个人作品集基础版本。

## 本地预览

```bash
npm.cmd install
npm.cmd run dev
```

然后打开：

```text
http://127.0.0.1:4177/
```

## 当前页面结构

1. 全屏首页 Hero：视频背景、大标题、导航栏、联系按钮
2. 个人经历：人物视频、个人介绍、联系方式、项目数据
3. 精选项目：大卡片作品图片
4. 个人优势：能力卡片
5. 底部联系：整屏收尾页

## 设计方向

- 暗色系
- 高级、克制、有科技感
- PC 端优先
- 版心约 1700px
- 后续根据截图和参考网站继续细化
